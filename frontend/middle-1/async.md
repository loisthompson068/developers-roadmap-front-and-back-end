* Почему для асинхронных действий не получится использовать `try..catch`?
    ```javascript
    try {
      setTimeout(function() { throw new Error('Async error!'); }, 1000);
    } catch (e) {
      console.log(e);
    }
    ```

* Что выведет данная конструкция:
    ```javascript
    for (var i = 0; i <= 3; i++){
      setTimeout(function(){ console.log(i); }, 0); 
    }
    ```
